# Personal Portfolio

A modern and responsive personal portfolio developed to showcase my projects, skills, and learning journey as a Software Development student.

## 🚀 Technologies

* HTML5
* CSS3

## 📖 About the Project

This portfolio was built as part of a Rocketseat practical challenge. The goal was to apply modern HTML and CSS concepts while creating a clean, responsive, and visually appealing personal website.

The website presents information about me, my skills, featured projects, and contact details in a professional and organized layout.

## ✨ Features

* Responsive design
* Modern and clean interface
* About Me section
* Projects showcase
* Skills section
* Contact links
* Smooth navigation

## 🎯 What I Learned

* Building responsive layouts
* CSS Flexbox
* CSS Grid
* Semantic HTML
* Component organization
* Better UI structuring
